Privacy Policy for Crypto Price Tracker Chrome Extension

Last updated: October 2024

1. Introduction

This privacy policy explains how our BitLiq Chrome Extension ("Extension") handles user data and privacy.

2. Data Collection

Our Extension does not collect, store, or transmit any personal information from its users.

3. API Usage

The Extension makes external API calls to CoinGecko to fetch cryptocurrency price data. These calls do not include any personal information.

4. Data Storage

All data displayed in the Extension is temporary and is not stored permanently. The data is refreshed each time the Extension is used.

5. Third-Party Services

We use CoinGecko's public API to provide cryptocurrency data. Please refer to CoinGecko's privacy policy for information on how they handle data.

6. Changes to This Privacy Policy

We may update our Privacy Policy from time to time. We will notify you of any changes by posting the new Privacy Policy on this page.
